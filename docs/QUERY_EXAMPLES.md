# Supported Query Examples

The Airfield Data Intelligence system supports 11 core query types. Below are example natural language queries for each type.

---

## 1. Taxi-In Performance by Aircraft Type

Analyze arrival taxi times from landing to gate.

```
- Compare average taxi times by aircraft type
- Show taxi time breakdown by aircraft type
- Show me taxi-in performance by aircraft type
- What are the average taxi-in times for different aircraft types?
- Compare taxi-in times by aircraft type
```

---

## 2. Taxi-Out Performance by Hour

Analyze departure taxi times across different hours.

```
- Show taxi-out times by hour
- What are the taxi-out times across different hours?
- Show me hourly taxi-out performance
- When is the surface most congested for taxi-out?
- Compare taxi-out times by time of day
```

---

## 3. Movement Area Occupancy

```
- How many aircraft are in the movement area?
- Show movement area occupancy
- Analyze ground saturation in movement area
- What is the movement area occupancy by hour?

```

---

## Tips for Best Results

1. **Be specific** - "Show taxi-in times by aircraft type" works better than "show times"
2. **Use keywords** - Include terms like "taxi", "runway", "gate", "delay", "aircraft type"
3. **Ask one question at a time** - Complex multi-part queries may be less accurate
4. **Use comparisons** - "Compare X by Y" format works well

---

## Query Response Format

Each query returns:
- **Natural language answer** - Human-readable summary
- **Data table** - Raw results (when applicable)
- **Chart visualization** - Bar/line charts for numeric data
